# AI Universal Hub

A personal AI platform integrating multiple AI models in one interface.

## Integrated AI
- ChatGPT
- Claude
- Gemini
- Perplexity

## Features
- Single dashboard for all AI
- Prompt library
- AI comparison
- Workflow automation

## Project Structure

frontend/ → user interface  
backend/ → API integrations  
prompts/ → reusable prompts  
docs/ → documentation
